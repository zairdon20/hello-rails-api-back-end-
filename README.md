## Built With

- Ruby
- Ruby on rails --api
- 💓

# Get Started
To get a local copy up and running follow these simple steps.

```bash
$ git clone https://github.com/omarsalem7/hello-rails-back-end
$ cd hello-rails-back-end
$ bundle install 
$ rake db:create 
$ rake db:migrate 
$ rails s 
```
